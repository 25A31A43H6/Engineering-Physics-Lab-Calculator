Overview

This is a basic calculator program written in C++. The program allows users to perform simple arithmetic operations such as:

Addition (+)
Subtraction (-)
Multiplication (*)
Division (/)

The calculator takes two numbers and an operator as input from the user and displays the result.

Features
User-friendly console interface
Supports four arithmetic operations
Handles division by zero errors
Uses switch statement for operation selection
Technologies Used
Programming Language: C++
Compiler: Any standard C++ compiler (G++, Turbo C++, VS Code, Code::Blocks, etc.)
Program Flow
Display calculator title
Read first number from the user
Read arithmetic operator
Read second number
Perform calculation using switch statement
Display result or error message
Code Concepts Used
Variables and Data Types
Input/Output (cin, cout)
Arithmetic Operators
switch-case
Conditional Statements (if-else)
